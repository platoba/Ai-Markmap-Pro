# Changelog - Ai-Markmap-Pro

## v2.0.0 - Deep Enhancement (2026-03-03)

### 🎯 New Features
- **History Management**: Local storage of generated mind maps with search and restore
- **Template System**: 10+ pre-built templates for different scenarios (brainstorming, project planning, learning notes, etc.)
- **Theme System**: Dark mode + 5 color schemes (Ocean, Forest, Sunset, Purple, Monochrome)
- **Advanced Export**: Support for Markdown, JSON, and PDF export
- **Share System**: Generate shareable URLs with encoded mind map data
- **Prompt Library**: 15+ built-in AI prompt templates for different use cases
- **Enhanced Shortcuts**: Extended keyboard shortcuts for power users
- **Auto-save**: Automatic saving of work in progress
- **Search & Filter**: Search through history and filter by date/tags
- **Batch Operations**: Export multiple versions at once

### 🔧 Improvements
- Optimized mobile experience with better touch gestures
- Improved SVG/PNG export quality
- Better error handling and user feedback
- Performance optimization for large mind maps
- Enhanced accessibility (ARIA labels, keyboard navigation)

### 🐛 Bug Fixes
- Fixed export filename generation
- Improved cross-browser compatibility
- Fixed mobile landscape mode issues

### 📚 Documentation
- Added comprehensive README with examples
- Included usage guide and API documentation
- Added contribution guidelines
